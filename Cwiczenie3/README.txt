* Zawartosc:
============

-W poleceniu zadane bylo uruchomienie programu przy pomocy zadania pierwszego-
ja umiescilem dla wygody wszystko w osobnym pliku.




Katalog zawiera program cwiczenie 3 sprawdzajacy dzialanie funkcji exec w C.
Aby uruchomic program nalezy uzyc polecenia:







-> Aby uruchomic program, nalezy wykonac komende:
       make run1



-> Aby wyczyscic zawartosc katalogu (usunac zbedne pliki), nalezy wykonac:
       make clean

-> W celu archiwizacji oraz kompresji katalogu nalezy wykonac:
       make tar
